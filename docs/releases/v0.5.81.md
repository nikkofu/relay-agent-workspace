# v0.5.81 - Phase 42 File Collaboration And Knowledge Metadata APIs

Date: 2026-04-21

## Summary

This release turns files into first-class collaborative knowledge objects with comments, shares, stars, and LLM/wiki-ready metadata.

## Shipped APIs

- `GET /api/v1/files/:id/comments`
- `POST /api/v1/files/:id/comments`
- `GET /api/v1/files/:id/shares`
- `POST /api/v1/files/:id/share`
- `POST /api/v1/files/:id/star`
- `GET /api/v1/files/starred`
- `PATCH /api/v1/files/:id/knowledge`

## File Knowledge Fields

File payloads now include:

- `knowledge_state`
- `source_kind`
- `summary`
- `tags`
- `comment_count`
- `share_count`
- `starred`

## Collaboration Behavior

Sharing a file creates a real message plus `MessageFileAttachment`, allowing file knowledge to propagate into channel/thread history.

## Windsurf Handoff

Start with Files page and file detail surfaces:

- comments list/create
- share into channel/thread
- star toggle / starred files
- knowledge metadata editing
