# Release v0.5.27

## Summary

This release adds artifact version history to Relay Agent Workspace so collaborative canvases can be inspected over time instead of behaving like opaque latest-only records.

## Delivered

- `artifact_versions` persistence
- latest artifact `version` field on artifact payloads
- `GET /api/v1/artifacts/:id/versions`
- `GET /api/v1/artifacts/:id/versions/:version`
- automatic version snapshot creation for:
  - artifact create
  - artifact update
  - AI canvas generation

## Why It Matters

This gives Gemini a clean backend contract for a real history panel in the canvas surface and sets up future rollback and diff work without changing the storage model again.
