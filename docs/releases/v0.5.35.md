# Release v0.5.35

## Summary

This release upgrades workspace search with typed suggestions and richer result payloads so Gemini can build a more useful command-palette style search experience.

## Delivered

- `GET /api/v1/search/suggestions?q=...`
- richer `GET /api/v1/search` payloads with snippets and match reasons

## Why It Matters

Relay now has the first step toward a real discovery layer instead of a plain keyword dump. This improves both Slack-style navigation and the AI-native workflow where users need to quickly recover channels, people, and prior context.

## Verification

- `cd apps/api && go test ./...`
- `cd apps/api && GOCACHE=$(pwd)/.cache/go-build go build ./...`
- `pnpm build`
