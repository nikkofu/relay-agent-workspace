# Release v0.5.46

## Summary

This release expands Relay's Slack-style workspace shell into real backend contracts for home, profiles, groups, status, workflows, and tools.

## Delivered

- `GET /api/v1/home`
- `GET /api/v1/users/:id`
- `PATCH /api/v1/users/:id/status`
- `GET /api/v1/user-groups`
- `GET /api/v1/user-groups/:id`
- `GET /api/v1/workflows`
- `GET /api/v1/tools`

## Why It Matters

Relay now has backend primitives for the surfaces that sit around chat:

- home can render a useful landing dashboard
- profile cards can grow into full personal profiles
- user groups can move from static UI concepts to real shared entities
- workflows and tools now have explicit registry endpoints for future agentic launchers

## Verification

- `cd apps/api && go test ./...`
- `cd apps/api && go build ./...`
- `pnpm build`
