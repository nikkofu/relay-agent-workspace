# v0.6.71 - Phase 74 Sales App Display Modes Plan

This planning release freezes the next Sales App refinement.

## Scope

- Replace Phase 73 display modes with `List`, `Card Grid`, `Kanban`, `Calendar`, and `Stat`.
- Keep one shared search/filter area above every mode.
- Add Calendar subviews: `day`, `week`, `month`.
- Project Calendar events from business time fields: `expected_close_date`, `order_date`, `due_date`, `last_activity_at`.
- Add Stat chart styles: `summary`, `bar`, `funnel`, `timeline`.

## Ownership

- Gemini owns backend/API/tests for mode aliases, calendar event projection, calendar/stat validation, and stat aggregate families.
- Windsurf owns Web/UI for the mode switcher, Card Grid, Calendar subviews, business time field selector, and Stat chart-style switcher.
- Codex owns contract questions, sequencing, final audit, collaboration docs, and release closeout.
