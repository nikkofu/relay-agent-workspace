# Relay Agent Workspace v0.5.72

This release hardens the workspace home payload and adds explicit draft deletion.

## Highlights

- `home` now returns the direct aliases the current dashboard actually reads
- drafts can now be explicitly deleted by scope
- Gemini has a clean path to finish home-data cleanup and composer discard/send cleanup

## Added Contracts

- `GET /api/v1/home`
  now includes:
  - `stats`
  - `recent_activity`
  - top-level `recent_artifacts`
- `DELETE /api/v1/drafts/:scope`

## Gemini Follow-Up

- map Home directly from the new aliases instead of partial nested lookups
- decide whether composer send/discard should call draft delete for true cleanup
