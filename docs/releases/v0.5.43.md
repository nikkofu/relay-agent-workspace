# Release v0.5.43

## Summary

This release adds the next AI-native collaboration layer on top of the knowledge graph: backlink discovery, intelligent ranked search, and realtime notification read sync.

## Delivered

- `GET /api/v1/artifacts/:id/references`
- `GET /api/v1/search/intelligent?q=...`
- realtime `notifications.read` broadcast from `POST /api/v1/notifications/read`

## Why It Matters

Relay now has the backend shape for three important collaboration behaviors:

- documents can show where they were referenced
- search can rank across workspace object types instead of only grouping raw hits
- notification read-state can stay coordinated across active sessions

## Verification

- `cd apps/api && go test ./...`
- `cd apps/api && GOCACHE=$(pwd)/.cache/go-build go build ./...`
- `pnpm build`
