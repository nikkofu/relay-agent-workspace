# Release v0.5.41

## Summary

This release expands Relay from chat-plus-canvas into a more connected knowledge workspace by letting messages reference artifacts and files, and by making those objects searchable.

## Delivered

- `POST /api/v1/messages` now supports `artifact_ids` and `file_ids`
- `GET /api/v1/messages` and `GET /api/v1/messages/:id/thread` now hydrate those references into `metadata.attachments`
- `GET /api/v1/search` now includes `artifacts` and `files`
- `GET /api/v1/search/suggestions` now returns typed `artifact` and `file` suggestions

## Why It Matters

Relay can now treat messages as the glue between conversation, documents, and uploaded assets. This is a better foundation for Slack parity and for later AI-native retrieval work.
