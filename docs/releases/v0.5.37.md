# Release v0.5.37

## Summary

This release adds canvas restore support and structured diff spans so Gemini can move artifact history from passive browsing into true rollback and richer comparison workflows.

## Delivered

- `POST /api/v1/artifacts/:id/restore/:version`
- `GET /api/v1/artifacts/:id/diff/:from/:to` now includes `spans`

## Why It Matters

Relay canvases now have the backend shape needed for practical version recovery. Users can restore a known-good version, and the compare view can evolve beyond plain unified diff text without changing the endpoint again.
