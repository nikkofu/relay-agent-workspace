# Release v0.5.51

## Summary

This release extends Relay's operational shell with editable people data, user group lifecycle APIs, richer file management, and workflow run realtime.

## Delivered

- `PATCH /api/v1/users/:id`
- `POST /api/v1/user-groups`
- `PATCH /api/v1/user-groups/:id`
- `DELETE /api/v1/user-groups/:id`
- `DELETE /api/v1/files/:id`
- expanded file filtering
- realtime `workflow.run.updated`

## Why It Matters

Relay now supports more than discovery surfaces around the workspace:

- people and identity data can be edited
- groups can be created and maintained
- files can be filtered and removed
- workflows can notify the UI as they start running

## Verification

- `cd apps/api && go test ./...`
- `cd apps/api && GOCACHE=$(pwd)/.cache/go-build go build ./...`
- `pnpm --filter relay-agent-workspace lint`
