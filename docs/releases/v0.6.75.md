# v0.6.75 - Phase 75 Atomic Mentions And Channel AI Replies

This release implements the next Slack-like and AI-native messaging slice.

## Scope

- Atomic user mention chips in channel/thread composers.
- Structured mention metadata from composer to backend persistence.
- Durable user type classification: `human`, `bot`, `ai`.
- Channel AI auto-replies when an AI/bot user is mentioned.
- DM-style AI reasoning, tool timeline, and token usage in channel replies.
- Realtime metadata preservation for `message.created`.
