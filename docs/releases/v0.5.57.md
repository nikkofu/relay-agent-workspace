# Release v0.5.57

## Summary

This release thickens the contracts behind profiles, workflow runs, and file governance so the existing Slack-like shell can render richer operational detail with less frontend adaptation.

## Delivered

- extended profile fields:
  - `pronouns`
  - `location`
  - `phone`
  - `bio`
- `workflow_run_steps`
- richer workflow run list/detail payloads
- richer file list/detail payloads
- dual `events` / `audit_history` file audit response

## Why It Matters

Relay now behaves more like a mature collaboration product:

- people data can feel more complete and personal
- workflow runs can show actual execution structure
- file governance screens can consume a stable payload shape directly
