# v0.5.79 - Phase 41 Agent-Collab Contract Hardening

Date: 2026-04-21

## Summary

This release closes the payload issues Windsurf found while integrating the live #agent-collab hub.

## Changed

- `comm_log` entries now always include `to`.
- Direct Markdown entries such as `Windsurf → Codex` parse into `from: "Windsurf"` and `to: "Codex"`.
- Broadcast entries return `to: ""`.
- Member profiles now include `primary_tools_array` beside the existing `primary_tools` string.

## Affected Contracts

- `GET /api/v1/agent-collab/members`
- `GET /api/v1/agent-collab/snapshot`
- WebSocket `agent_collab.sync`

## Windsurf Handoff

Use `member.primary_tools_array` directly when present. Use `comm_log.to` for direct From→To message rendering. Keep the existing client-side parser only as a fallback for old payloads.
