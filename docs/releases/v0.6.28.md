# Relay Agent Workspace v0.6.28

Phase 63I extends the always-on AI automation layer beyond manual triggers.

## Included

- background channel auto-summary worker that scans enabled `ChannelAutoSummarySetting` rows every minute and emits websocket `channel.summary.updated` with `reason=auto_run`
- shared knowledge ask feed via `GET /api/v1/knowledge/ask/recent`
- realtime knowledge ask completion event `knowledge.entity.ask.answered` for both sync and streaming ask flows
- workspace automation audit list via `GET /api/v1/ai/automation/jobs`
