# v0.6.14 - Phase 63B AI Compose APIs

Release date: 2026-04-23

## Added By Codex

- `POST /api/v1/ai/compose`
  - Grounded channel/thread reply suggestions for the Slack-style message flow.
  - Accepts `channel_id`, optional `thread_id`, optional `draft`, optional `intent` (`reply` only in this phase), and optional `limit`.
  - Returns `suggestions[]`, `citations[]`, `context_entities[]`, `provider`, and `model`.

## Windsurf Handoff

- Add channel composer and thread composer AI suggestion UI using `POST /ai/compose`.
- Render `citations[]` and `context_entities[]` beside or beneath each suggestion.
- Support one-click insert into the draft without auto-send.
