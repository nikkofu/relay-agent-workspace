# Relay Agent Workspace v0.6.32

This release completes Phase 64B backend support for Windsurf's unified activity rail.

## Added

- `GET /api/v1/activity/feed`
- Unified activity feed contract fields:
  - `event_type`
  - `workspace_id`
  - `actor_id` / `actor_name`
  - `channel_id` / `channel_name`
  - `dm_id`
  - `entity_id` / `entity_title` / `entity_kind`
  - `title`, `body`, `link`, `occurred_at`, `meta`
- Current feed sources:
  - `message`
  - `file_uploaded`
  - `schedule_booking`
  - `compose_activity`
  - `knowledge_ask`
  - `automation_job`
