# Release v0.5.29

## Summary

This release adds artifact diff support so Relay can compare two saved canvas versions without requiring the frontend to reconstruct changes on its own.

## Delivered

- `GET /api/v1/artifacts/:id/diff/:from/:to`
- unified diff output
- change summary with added and removed line counts
- raw before/after content payloads for flexible rendering

## Why It Matters

Gemini now has a clean backend contract for a dedicated comparison view in the canvas history UI. This keeps diff logic centralized and prepares the system for richer review workflows later.
