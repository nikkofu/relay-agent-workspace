# v0.6.16 - Phase 63C AI Compose Stream And Feedback APIs

Release date: 2026-04-23

## Added By Codex

- `POST /api/v1/ai/compose/stream`
  - SSE endpoint for grounded channel/thread reply suggestions.
  - Emits `start`, `suggestion.delta`, `suggestion.done`, `done`, and `error`.
- `POST /api/v1/ai/compose/:id/feedback`
  - Persists per-suggestion `up`, `down`, or `edited` feedback with upsert semantics by `(compose_id, user_id)`.

## Windsurf Handoff

- Switch the composer suggestion popover from sync compose to `POST /ai/compose/stream` when SSE is available.
- Add thumbs-up / thumbs-down / edited controls per suggestion using `POST /ai/compose/:id/feedback`.
- Keep `POST /ai/compose` as a compatibility fallback.
