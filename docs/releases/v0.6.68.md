# v0.6.68 - Phase 73 Business App Multiview Sales Plan

This planning release freezes the next Super App Workspace slice.

## Scope

- Add App Hub / Sales App as the first reusable business-app instance.
- Build one multiview business page that can display the same Sales Orders data as search, list, calendar, kanban, and stats.
- Add a visible workspace menu link for Sales App.
- Keep AI-native actions metadata-only for this phase.

- Windsurf owns Web/UI for `/workspace/apps/sales`, reusable business app shell, mode switcher, Sales Order views, and App Hub/menu navigation.
