# v0.5.3 Release Notes

## Summary

This release unifies the latest Gemini frontend delivery with the next Codex backend wave by shipping inbox and mentions APIs on top of the completed channel management work.

## Included From Gemini

- Phase 10 frontend integration for:
  - channel info
  - member lists
  - topic and purpose editing

## Added By Codex

- `GET /api/v1/inbox`
- `GET /api/v1/mentions`

## API Behavior

- inbox returns aggregated collaboration signals for the current user:
  - mentions
  - thread replies
  - reactions
  - DM activity
- mentions returns the direct-mention subset only

## Gemini Handoff

- next UI target:
  - inbox page or panel
  - mentions surface
- both endpoints return:
  - `{ items: [...] }`
- item fields:
  - `id`
  - `type`
  - `user`
  - optional `channel`
  - optional `message`
  - optional `target`
  - `summary`
  - `occurred_at`

- `cd apps/api && go test ./...`
- `cd apps/api && GOCACHE=$(pwd)/.cache/go-build go build ./...`
- `pnpm build`
