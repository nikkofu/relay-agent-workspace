# v0.6.12 - Phase 63A Knowledge Ask And Share APIs

Release date: 2026-04-22

## Added By Codex

- `POST /api/v1/knowledge/entities/:id/ask`
  - Accepts `question`, optional `provider`, and optional `model`.
  - Grounds against entity refs, timeline events, and linked entities.
  - Returns `answer`, `citations[]`, and entity metadata.
- `POST /api/v1/knowledge/weekly-brief/:id/share`
  - Returns a share payload for a persisted weekly brief snapshot.
  - Weekly brief responses now include a stable snapshot `id`.
- Websocket `knowledge.entity.brief.changed`
  - Broadcast when a cached entity brief becomes stale after new refs, events, entity edits, or auto-link flows.

## Windsurf Handoff

- Add an entity-detail Ask AI module using `POST /knowledge/entities/:id/ask`.
- Add weekly digest sharing using `brief.id` from current weekly brief state and `POST /knowledge/weekly-brief/:id/share`.
- Listen for `knowledge.entity.brief.changed` and surface a Refresh / Regenerate CTA when a cached brief is stale.
