# Relay Agent Workspace v0.6.22

Release date: 2026-04-23

## Summary

Phase 63F adds always-on knowledge automation contracts for channel summaries and collaborative AI compose signals.

## Added APIs

- `GET /api/v1/channels/:id/knowledge/auto-summarize`
- `PUT /api/v1/channels/:id/knowledge/auto-summarize`
- `POST /api/v1/channels/:id/knowledge/auto-summarize`

`PUT` persists per-channel settings:

- `is_enabled`
- `window_hours`
- `message_limit`
- `min_new_messages`
- `provider`
- `model`

`POST` runs a rolling channel summary over recent messages, persists the result into the existing channel `AISummary`, updates `last_run_at` and `last_message_at`, and returns:

- `setting`
- `summary`

## Added Realtime Events

- `channel.summary.updated`
- `knowledge.compose.suggestion.generated`

## AI Compose Additions

Schedule intent responses now include additive `proposed_slots[]`:

- `starts_at`
- `ends_at`
- `duration_minutes`
- `timezone`
- `attendee_ids`
- `reason`

## Windsurf Handoff

- Add channel auto-summarize controls using `GET|PUT|POST /channels/:id/knowledge/auto-summarize`.
- Refresh channel summary cards on websocket `channel.summary.updated`.
- Render `compose.proposed_slots[]` as calendar chips for schedule intent.
- Optionally surface `knowledge.compose.suggestion.generated` in co-drafting or agent-collab activity.
